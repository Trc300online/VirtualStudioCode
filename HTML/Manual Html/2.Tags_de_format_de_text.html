2.Els tags de format de text

  1. Etiquetes de capçalera (heading) <h1> … <h6>

    Etiquetes que es solen utilitzar per remarcar text especial que defineix els apartats del nostre web. Hi ha 6 nivells d'etiquetes de capçalera

  2. Etiqueta de paràgraf <p>

    La majoria del contingut d'una web és text. Tots els paràgrafs de text que apareguin al nostre web han d'anar a dins l'etiqueta <p>. Un paràgraf sempre comença a una línia nova.

  3. Generador de paràgrafs Lorem Ipsum

    Lorem Ipsum és un generador de paràgrafs que s'utilitza per posar contingut a la nostra web per poder veure com va quedant la nostra web si encara no tenim el contingut real. posar "Lorem" dins l'editor de html

  4. Etiquetes de format

    Les etiquetes de format serveixen per mostrar text que tingui un caràcter especial.

    • <b> - Text en negreta (bold)
    • <strong> - Text important o amb paraules clau
    • <i> - Text en cursiva (italic)
    • <em> - Text emfatitzat (emphasized)
    • <mark> - Text resaltat (com amb rotulador)
    • <small> - Anotacions menors
    • <del> - Text tatxat (deleted)
    • <ins> - Text subratllat (inserted)
    • <sub> - Subíndex (subscript)
    • <sup> - Superíndex (superscript)
