<!DOCTYPE html>
Informa al navegador web que el document HTML està format amb l'especificació d'HTML5
<html>
És l'etiqueta arrel del document HTML. És la que conté les dues seccions principals: <head> i <body>.
<head>
Secció del document HTML on s'especifiquen les metadades del nostre document. El contingut d'aquesta secció no es mostra directament a la nostra web.
<body>
Secció del document HTML on posarem totes les etiquetes de contingut. El que es posa a dins aquesta secció és el que apareixerà a la nostra web.
Etiquetes de capçalera (heading) <h1> … <h6>
Etiquetes que es solen utilitzar per remarcar text especial que defineix els apartats del nostre web. Hi ha 6 nivells d'etiquetes de capçalera
Etiqueta de paràgraf <p>
La majoria del contingut d'una web és text. Tots els paràgrafs de text que apareguin al nostre web han d'anar a dins l'etiqueta <p>. Un paràgraf sempre comença a una línia nova.
Generador de paràgrafs Lorem Ipsum
Lorem Ipsum és un generador de paràgrafs que s'utilitza per posar contingut a la nostra web per poder veure com va quedant la nostra web si encara no tenim el contingut real. posar "Lorem" dins l'editor de html
Etiquetes de format
Les etiquetes de format serveixen per mostrar text que tingui un caràcter especial.
  • <b> - Text en negreta (bold)
  • <strong> - Text important o amb paraules clau
  • <i> - Text en cursiva (italic)
  • <em> - Text emfatitzat (emphasized)
  • <mark> - Text resaltat (com amb rotulador)
  • <small> - Anotacions menors
  • <del> - Text tatxat (deleted)
  • <ins> - Text subratllat (inserted)
  • <sub> - Subíndex (subscript)
  • <sup> - Superíndex (superscript)
Links (anchor)
Una de les eines més potents d'HTML és l'utilització de links, ja sigui a un altre plana web o a parts del nostre document. El format de l'etiqueta per fer un link és el següent: <a href="url">link text</a>
Comentaris
L'ús de comentaris dins el nostre document HTML és molt útil. Ens ajuden a entendre el que hi ha a les diferents parts del document. L'explorador ignora els comentaris i no es mostren a la pantalla, però fan el codi més legible per al programador. Amb la seguent estructura <!-- (text)-->
